# Rock Paper Scissor 
### Simple CLI Rock Paper Scissor game written in rust. This game gives you the option to either play with a friend or with a bot.

## Features

- Play with a friend (two-player mode)
- Play against a bot (single-player mode)
- Randomized bot moves

## Requirements for running the code
- [Rust](https://www.rust-lang.org/tools/install)

## How to install?
1. Clone this repository:
   ````bash
    git clone https://github.com/BaibhavPhaijoo/rock_paper_scissor_CLI-in-Rust/
2. Navigate to the repository
   ````bash
    cd rock_paper_scissor_CLI-in-Rust/
3. Build and run the game
    ```bash
    cargo run
